# Go Blockchain Core Suite
高性能、原生Go实现的区块链底层核心工具集，覆盖区块链全场景开发需求，无第三方依赖、开箱即用、可直接用于学习与工程落地。

## 包含核心模块
本仓库集成12大区块链核心功能，全部采用Go语言原生开发：
1. UTXO账本模型（比特币底层）
2. PoS权益证明共识引擎
3. 非对称加密密钥对生成器
4. 默克尔树根哈希计算
5. P2P节点消息广播
6. 区块奖励减半算法
7. 交易签名与验签
8. 创世区块生成
9. 钱包余额扫描统计
10. 区块同步高度校验
11. ERC20代币标准实现
12. 区块数据加密

## 适用场景
- 区块链底层学习
- 公链/联盟链开发
- 加密钱包、共识引擎、P2P网络
- 交易验签、数据安全、代币发行
